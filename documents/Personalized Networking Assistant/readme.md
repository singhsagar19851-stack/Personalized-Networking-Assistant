# Personalized Networking Assistant

## Overview

A full-stack AI-powered networking assistant that helps users generate conversation starters for professional networking events.

## Features

- Event theme extraction using BART Large MNLI
- AI conversation generation using GPT-2
- Wikipedia fact checking
- Conversation history
- Feedback logging
- FastAPI backend
- Streamlit frontend
- Unit & Integration tests

## Tech Stack

- Python
- FastAPI
- Streamlit
- Hugging Face Transformers
- GPT-2
- Facebook BART MNLI
- Pytest
